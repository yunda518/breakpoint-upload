<h2>breakpoint-upload</h2>
<h2>一、基本介绍：</h2>
  Golang 实现网页版断点续传工具 —— 通过前端页面交互配合后端逻辑，实现了文件分片传输与断点记录功能。即便夜间上传过程中出现中断，次日也能从断点处继续传输，极大提升了大文件上传的便捷性与稳定性。
<h2>二、技术特点</h2>
<b>前端技术点：</b><br>
1、纯前端实现复杂功能 ：不依赖第三方上传库，使用原生JavaScript实现分块上传<br>
2、无缝断点续传 ：用户体验流畅的暂停/恢复功能<br>
3、现代化UI ：符合当代设计趋势的视觉体验<br>
4、完整的状态管理 ：从文件选择到上传完成的全流程状态跟踪（-显示上传进度百分比-实时计算并显示上传速度-动态估算剩余上传时间）<br>
5、离线能力 ：利用localStorage实现部分离线功能<br><br>
<b>后端技术点：</b><br>
1、大文件支持 ：通过分块上传处理大文件，最大支持 20GB 文件（分块上传机制 ：设置10MB为固定分块大小，减少单个请求压力）<br>
2、断点续传 ：利用本地存储保存上传状态，支持中断后继续上传<br>
3、并发安全 ：后端使用互斥锁保护共享状态<br>
4、错误恢复 ：前端实现上传失败自动重试机制<br>
5、用户友好 ：提供实时上传状态和进度反馈<br>
